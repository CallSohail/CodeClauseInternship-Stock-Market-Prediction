# Stock Price Prediction with Streamlit

## Overview

This project utilizes Python and various libraries such as Pandas, NumPy, yfinance, Scikit-learn, Plotly, and Streamlit to create an interactive stock price prediction application. The application allows users to select a stock ticker from a predefined list and visualizes the stock's historical data, moving averages, and provides predictions for future stock prices.

## Features

- **Stock Selection**: Users can choose a stock ticker from a list of popular options.
- **Data Visualization**: The application displays the historical closing prices of the selected stock along with 100-day and 200-day moving averages for trend analysis.
- **Data Description**: Provides a summary of the data including statistics such as mean, standard deviation, and more.
- **Price Prediction**: Utilizes a trained machine learning model to predict future stock prices based on historical data.
- **Interactive Charts**: All visualizations are interactive and allow users to zoom in, pan, and hover over data points for more information.
- **User-Friendly Interface**: The Streamlit application provides an easy-to-use interface for users to explore stock data and predictions.
